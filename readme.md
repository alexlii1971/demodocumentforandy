# 商业航天可回收火箭全产业链标的投研分层对比总表（专业机构版）

## 投研分层核心逻辑说明

本表完全遵循**专业机构投研决策框架**，以「业绩确定性>技术路线适配性>合规安全性>弹性空间>兑现周期」为核心权重，将全部标的划分为5个严格分层，每层先明确**定位、筛选标准、操作规则**，再对应展示该层标的全维度对比表，彻底解决此前标的混排、分层逻辑不落地的问题。

- 数据来源：上市公司行政处罚决定书、官方澄清公告、互动易董秘答复、券商航天产业深度研报、航天院所招标公示、船舶集团官方新闻
- 免责声明：本表仅为产业链基本面客观梳理，不构成任何投资建议，商业航天存在试验失败、订单延期、行业内卷、政策波动多重不确定性

## 第一层：核心配置层（底仓配置，优先布局）

### 【分层定位】

全产业链最优质标的，兼具**全技术路线通用、真实耗材复购逻辑、高业绩确定性、无重大合规瑕疵**，是可回收火箭产业化放量的核心受益资产，适合作为商业航天板块底仓长期配置。

### 【筛选标准】

1. 配套产品适配着陆腿/网式回收全路线，无技术路线淘汰风险
2. 航天业务营收占比高，在手确定性订单充足，业绩兑现周期明确
3. 具备真实耗材复购属性，发射频次提升直接带动业绩持续增长
4. 无证监会立案、行政处罚、监管问询等重大合规风险

回调至合理估值区间分批建仓，长期持有，核心跟踪复飞进度、季度订单增速，不做短线频繁交易。

## 第二层：波段博弈层（仅阶段性行情，不长期持有）

### 【分层定位】

细分赛道具备刚需属性，短期事件催化弹性充足，但存在**客户集中、估值过高、业务分散、现金流承压、合规瑕疵**等单一硬伤，仅适合板块情绪上行、火箭发射催化窗口期参与，不适合长期持有。

### 【筛选标准】

1. 配套产品为可回收火箭细分环节刚需，具备短期订单放量逻辑
2. 存在明确的业绩制约因素，或估值已透支远期预期，或客户集中度极高
3. 无重大行政处罚、立案调查等致命合规风险（铂力特除外，单独标注风险）

### 【操作规则】

催化落地前埋伏，催化兑现后止盈离场，严格设置止损，不恋战，不追高。

## 第三层：基建配套层（长线低弹性，大资金底仓）

### 【分层定位】

海上回收平台专属配套标的，仅适配**网式回收单一路线**，订单为大额一次性项目，交付周期极长，业绩兑现节奏慢，仅适合长线产业资金布局，不适合短线博弈。

### 【筛选标准】

1. 配套产品为海上回收平台核心建造/设计/集成环节，单项目价值量高
2. 技术路线绑定网式回收，民营火箭着陆腿回收路线无配套需求
3. 具备央企船舶集团资质壁垒，短期无替代厂商

### 【操作规则】

仅在估值低位布局，博弈未来多艘海上回收平台批量建造预期，长期持有，不参与短线波动。
